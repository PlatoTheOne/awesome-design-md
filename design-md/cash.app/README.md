# Cash App Inspired Design System

[DESIGN.md](https://github.com/PlatoTheOne/awesome-design-md/blob/main/design-md/cash.app/DESIGN.md) extracted from the public [Cash App Brand Guidelines](https://design.cash.app/color). This is not the official design system. Colors, fonts, and spacing may not be 100% accurate. But it's a good starting point for building something similar.
